# Debugging the Thinking

Most engineers debug the code.

Great engineers debug the thinking behind the code.

Many bugs originate from:
- Wrong assumptions
- Unclear requirements
- Missing edge cases
- Weak system thinking

Code is usually the final step in a long chain of decisions.

If the thinking is wrong, the code will eventually fail.
